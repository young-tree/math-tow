#### 1.什么是偏导

- \(\boldsymbol{\dfrac{d}{dx}}\)：**一元函数普通导数**，函数只有 x 一个变量，所有变量都跟着 x 一起变
- \(\boldsymbol{\dfrac{\partial}{\partial x}}\)：**多元函数偏导数**，函数有 \(x,y,z\) 等多个变量，只让 x 变化，其余变量固定为常数
- 例：\(z=x^2+3xy\)，对 x 偏导：\(\dfrac{\partial z}{\partial x}=2x+3y\)
  - 其实就是对 \(x^2\) 求导再对 \(3xy\) 求导，y是常数

#### 2.什么是可微

- 在曲线某一个点附近取很小一段，改用这个点上的切线（一条笔直的直线）去代替这段曲线做估算
- 一元函数可微：\(\Delta y = A\Delta x + o(\Delta x)\)
- \(o(\Delta x)\) **不是某一个固定函数**，而是一类函数的统称，若函数 \(\alpha(\Delta x)\) 满足：\(\lim_{\Delta x \to 0}\frac{\alpha(\Delta x)}{\Delta x}=0\)，我们就记 \(\boldsymbol{\alpha(\Delta x)=o(\Delta x)}\)

  - \(o(\Delta x)\) **是 \(\Delta x\) 的高阶无穷小**
  - 设 \(\Delta x\to0\)：
  - \(\alpha_1=(\Delta x)^2\)：\(\displaystyle \lim_{\Delta x\to0}\frac{(\Delta x)^2}{\Delta x}=\lim \Delta x=0\)，所以 \((\Delta x)^2=o(\Delta x)\)
  - \(\alpha_2=5(\Delta x)^3\)：\(\displaystyle \lim_{\Delta x\to0}\frac{5(\Delta x)^3}{\Delta x}=0\)，所以 \(5(\Delta x)^3=o(\Delta x)\)
  - \(\alpha_3=\sin(\Delta x)^2\)：同样极限为 0，也属于 \(o(\Delta x)\)
    - 当 \(t\to0\) 时，等价无穷小公式：\(\sin t \sim t\)。令 \(t=(\Delta x)^2\)，当 \(\Delta x\to0\) 时，\(t=(\Delta x)^2\to0\)，因此：\(\sin\left((\Delta x)^2\right) \sim (\Delta x)^2\)
    - 替换后极限变为：\(\lim_{\Delta x \to 0}\frac{(\Delta x)^2}{\Delta x} =\lim_{\Delta x \to 0}\Delta x =0\)
- 如果没有 \(o(\Delta x)\)，等式根本不成立

  - 举个最简单的例子：\(y=x^2\)，取某点 x，给 x 增加一小段 \(\Delta x\)
  - 真实变化量：\(\Delta y=(x+\Delta x)^2 - x^2 = 2x\cdot\Delta x + (\Delta x)^2\)
  - 对比标准式 \(\Delta y=A\Delta x + o(\Delta x)\)：这里 \(A=2x\)，误差项就是 \((\Delta x)^2\)，而 \((\Delta x)^2=o(\Delta x)\)

  - 只写 \(\Delta y=A\Delta x\) 是错的，左边是真实变化，右边只是直线近似，二者天生差一块
  - 必须把差值 \((\Delta x)^2\) 写进等式，才能让左右严格相等，这一块差值就用 \(o(\Delta x)\) 统一表示
  - \(o(\Delta x)\) **是 \(\Delta x\) 的高阶无穷小**，这个约束是 “可微” 的灵魂
- A是什么？\(A=f'(x)\)，为什么？
  - 函数在 x 处可微的定义：Δy=A⋅Δx+o(Δx)，A 是和 \(\Delta x\) 无关的常数，\(o(\Delta x)\) 是比 \(\Delta x\) 高阶无穷小，满足：\(\lim_{\Delta x \to 0}\frac{o(\Delta x)}{\Delta x}=0\)
  - 导数 \(f'(x)\)​ 的定义（差商极限）：\(f'(x)=\lim_{\Delta x \to 0}\frac{\Delta y}{\Delta x}\)
  - 把可微式子代入导数极限证明 \(A=f'(x)\)
    - 对 \(\Delta y = A\Delta x + o(\Delta x)\) 两边同除以 \(\Delta x\)：\(\frac{\Delta y}{\Delta x}=A + \frac{o(\Delta x)}{\Delta x}\)
    - 两边同时取 \(\Delta x \to 0\) 的极限：\(\lim_{\Delta x \to 0}\frac{\Delta y}{\Delta x} =\lim_{\Delta x \to 0}\left(A + \frac{o(\Delta x)}{\Delta x}\right)\)
    - 左边：就是导数 \(f'(x)\)，右边第一项：常数 A，极限还是 A，右边第二项：根据高阶无穷小定义，\(\displaystyle \lim_{\Delta x\to0}\frac{o(\Delta x)}{\Delta x}=0\)
    - 化简得到：\(\boldsymbol{f'(x)=A}\)


#### 3.什么是微分，什么是导数

- 把式子\(\Delta y = A\Delta x + o(\Delta x)\)里面线性主部 \(A\Delta x\) 单独取名叫做微分 dy
- 又因为 \(A=f'(x)\)、规定自变量微分 \(dx=\Delta x\)，所以标准写法：\(dy = f'(x) dx\)

- 导数：\(\boldsymbol{\dfrac{dy}{dx}}\)
- \(y=x^2\)，导数 \(f'(x)=2x\)，微分写作：\(dy = 2x \cdot dx\)

#### 4.可微和可导

- 一元函数：定理：**可微 ⇔ 可导，二者完全等价，没有区别**
  - **可导**：只看变化率极限：\(f'(x)=\lim_{\Delta x\to0}\frac{\Delta y}{\Delta x}\)。描述：自变量微小变化时，y 的瞬时变化速率存在、是个有限数字。
  - **可微**：看增量拆分结构：\(\Delta y = A\Delta x + o(\Delta x)\)。描述：y 的真实变化可以拆成一条直线的变化 + 极小误差。
  - 前面证明过：这里的 A 正好等于导数 \(f'(x)\)，所以一元里两者是一回事，只是两种说法
    - 举例：\(y=x^2\)，处处可导，因此处处可微
    - y=|x| 在 \(x=0\) 不可导，因此在 \(x=0\) 也不可微
- 二元 / 多元函数（\(z=f(x,y)\)，x、y 两个自变量）
  - 不等价！强弱关系：**可微 \(\implies\) 两个偏导数都存在；反过来偏导存在 ≠ 可微**
    1. 可微（曲面顺滑，局部能用平面贴合）→ 一定有 \(\frac{\partial z}{\partial x},\frac{\partial z}{\partial y}\)
    2. 仅仅两个偏导数都存在 → **不能推出可微**（曲面可能有尖沟，局部贴不上平面）

#### 5.隐函数求导公式

- 一元隐函数（\(F(x,y)=0\)，\(y=y(x)\)），y 是关于 x 的函数 \(y=y(x)\)
  - 即 F 是以 x、\(y(x)\) 为中间变量的一元复合函数
  - 复合函数链式求导法则：对等式 \(F\big(x,\,y(x)\big) = 0\) 两边同时关于 x 求导
  - 左边求导：二元复合函数对自变量 x 的全导数，根据链式法则：\(\frac{dF}{dx} = \frac{\partial F}{\partial x} \cdot \frac{dx}{dx} + \frac{\partial F}{\partial y} \cdot \frac{dy}{dx}\)
    - 这个式子是怎么来的？对任意二元函数 \(F(x,y)\)，全微分定义：\(dF = \frac{\partial F}{\partial x} dx + \frac{\partial F}{\partial y} dy\)
    - 等式两边同时除以 dx，把全微分等式每一项都除以自变量微分 dx：\(\frac{dF}{dx} = \frac{\partial F}{\partial x} \cdot \frac{dx}{dx} + \frac{\partial F}{\partial y} \cdot \frac{dy}{dx}\)

  - 右边求导：常数求导都是0，0求导也是0
  - \(\dfrac{\partial F}{\partial x}\) 简写为 \(F_x\)，\(\dfrac{\partial F}{\partial y}\) 简写为 \(F_y\)
  - \(\boldsymbol{F_x + F_y \cdot \frac{dy}{dx} = 0}\)，\(\frac{dy}{dx} = -\frac{F_x}{F_y}\)
  - 举例：方程 \(x^2+y^2=1\)，令 \(F(x,y)=x^2+y^2-1\)
    - \(F_x=\dfrac{\partial F}{\partial x}=2x\)，\(F_y=\dfrac{\partial F}{\partial y}=2y\)
    - 对x求偏导，\(y^2-1\)是常数，求导数直接是零。对y求偏导，\(x^2-1\)是常数，求导数直接是零
    - 代入红框公式：\(2x + 2y \cdot \frac{dy}{dx} = 0\)，移项化简得 \(\dfrac{dy}{dx}=-\dfrac{x}{y}\)

- 二元隐函数\(F(x,y,z)=0\)，\(z=z(x,y)\)
  - 偏导公式：\(\boldsymbol{ \frac{\partial z}{\partial x} = -\frac{F_x}{F_z},\quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z} \quad (F_z \neq 0) }\)
  - 举例：\(e^z - xyz = 0\)，设\(F(x,y,z)=e^z-xyz\)
    - \(F_x=-yz,\ F_y=-xz,\ F_z=e^z-xy\)
    - \(F_x\)，对x求偏导，y和z都看作是常数，对y对z求偏导同理
    - \(\frac{\partial z}{\partial x}=-\frac{-yz}{e^z-xy}=\frac{yz}{e^z-xy}\)
